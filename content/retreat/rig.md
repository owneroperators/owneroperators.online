The point of the trip is to come home with recordings. Full chain in `gear.md`; these
are the ones that end the trip if forgotten:

- [ ] **Backup drive(s)** — record to local disk; Wi-Fi is not a backup plan
- [ ] Chargers — phones, laptop, interface power, camera
- [ ] Audio interface + enough cables (XLR ×6+, ¼" ×6+, USB/Thunderbolt)
- [ ] Headphones ×2–3 + a splitter, for tracking
- [ ] **Clip lights / a lamp** — no blinds in the loft; for night sessions
- [ ] Offline copy of the smartlock code + host contact
- [ ] Camera — this is a band-content trip too (EPK / socials)

> Rental rules: keep it quiet-friendly. Loud stuff during daytime lake hours; DI / amp-sims and reamp later once the kids are down.
